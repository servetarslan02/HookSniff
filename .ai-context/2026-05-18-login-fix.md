# 2026-05-18 — Login & Pricing Bug Fix

## Sorunlar

### 🔴 Login "Login failed" hatası
**Root cause:** Rate limit 429 response body'si boştu.
- `rate_limit_middleware` 429 döndürüyor ama `Body::empty()` → frontend JSON parse edemiyor → generic "Login failed" gösteriyor
- Aslında "Rate limit exceeded" olmalıydı

**İkinci sorun:** `extract_key()` fonksiyonu `TRUST_PROXY_HEADERS` ayarlı olmadığında tüm auth'suz isteklere `"unknown"` key'i veriyordu. Tüm kullanıcılar 100/dakika limiti paylaşıyordu.

**Üçüncü sorun:** Upstash Redis 500K/500K request limit'i dolmuş. Cloud Run Redis'e bağlanamıyor, in-memory rate limiter'e düşüyor. Redis Stream `hooksniff:events` quota'yı tüketiyor olabilir.

### 🔴 Landing page fiyatlar "Free/mo"
**Root cause:** `usePlans.ts`'de `formatPrice()` API çağrısı başarısız olunca (rate limit → 429 → plans boş) → `getPlan(id)` undefined → `getPlanPrice` 0 döner → "Free" gösterir.

### 🔴 Dashboard billing PlanCards yanlış fallback fiyatları
`PlanCards.tsx`'de fallback fiyatlar $14/$29/$99 idi, doğru değerler $29/$49.

## Yapılan Düzeltmeler

### rate_limit.rs
1. **429 body eklendi** — `Content-Type: application/json` ile `{"error": {"code": "RATE_LIMIT_EXCEEDED", "message": "..."}}` döndürüyor
2. **extract_key düzeltildi** — Artık `{ip}:{path}` formatında key kullanıyor. X-Real-IP ve X-Forwarded-For header'larından IP çıkarıyor

### usePlans.ts
- `getPlanPrice()` fonksiyonuna hardcoded fallback fiyatları eklendi (startup: $29, pro: $49)

### PlanCards.tsx
- Fallback aylık fiyatlar: $14/$29 → $29/$49
- Fallback yıllık fiyatlar: 168/278 → 278/470

## Yapılması Gerekenler (Servet)

1. **Cloud Build tetikle:**
   ```bash
   gcloud builds submit --config cloudbuild.yaml . --project=hooksniff-app
   ```

2. **Upstash Redis quota:**
   - Free plan 500K request/month dolmuş
   - Plan yükselt veya yeni instance oluştur
   - Cloud Run'daki `REDIS_URL` secret'ını güncelle

3. **TRUST_PROXY_HEADERS ekle:**
   cloudbuild.yaml'daki `--set-env-vars` listesine `TRUST_PROXY_HEADERS=true` ekle
