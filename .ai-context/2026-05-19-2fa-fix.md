## 2FA Akışı (Düzeltme Sonrası)

### Login + 2FA:
1. Kullanıcı email+password girer → POST /auth/login
2. Backend `requires_2fa: true` + `temp_token` döndürür (5 dk geçerli)
3. Frontend 2FA step gösterir (TOTP input veya backup code)
4. Kullanıcı kodu girer → POST /auth/2fa/verify
5. Backend JWT token döndürür → dashboard'a yönlendirme

### 2FA Enable:
1. Settings → 2FA → Enable → password doğrulama
2. QR code + secret gösterilir
3. TOTP kodu girilir → POST /auth/2fa/confirm
4. 8 backup code gösterilir

### 2FA Disable:
1. Settings → 2FA → Disable → password girilir
2. POST /auth/2fa/disable → 2FA kapatılır
