DEPLOYMENT WORKFLOW (UPDATED)
═══════════════════════════════════════════════════════════════════

1. Prepare environment:
   cp deploy/env.production.example .env.production
   # Fill all [REQUIRED] fields (see GCP_DEPLOYMENT_GUIDE.txt)

2. Deploy to GCP:
   chmod +x deploy/gcp-deploy.sh
   ./deploy/gcp-deploy.sh

3. The script will automatically:
   ✅ Validate all environment variables
   ✅ Create GCP project resources (Artifact Registry, APIs)
   ✅ Sync secrets to Google Secret Manager
   ✅ Build Docker images (API, Worker, Dashboard)
   ✅ Push to Artifact Registry
   ✅ Deploy all services to Cloud Run
   ✅ Output service URLs

4. Verify deployment:
   bash deploy/troubleshoot-gcp.sh all

EXTERNAL SERVICES REQUIRED (SIGN UP FIRST)
═══════════════════════════════════════════════════════════════════

Before deployment, sign up and get credentials for:

1. Database: Neon (PostgreSQL)
   → https://console.neon.tech
   → Get: DATABASE_URL

2. Cache: Upstash (Redis)
   → https://console.upstash.com
   → Get: REDIS_URL

3. Payments: Polar.sh
   → https://polar.sh
   → Get: POLAR_ACCESS_TOKEN, POLAR_WEBHOOK_SECRET

4. Email: Resend
   → https://resend.com
   → Get: RESEND_API_KEY

5. OAuth (optional): Google & GitHub
   → https://console.cloud.google.com/apis/credentials
   → https://github.com/settings/apps
